# Monet

Monet 是一个基于 Google 动态颜色主题开发的 .NET Material You 风格颜色库

参考了 [libmonet](https://cs.android.com/android/platform/superproject/main/+/main:external/libmonet) 这个项目，让 .NET 项目也可以拥有像 Google 动态颜色主题一样的效果，根据背景色选择主题色
